#Survey Form

A simple and responsive **Survey Form** built with HTML and CSS.  
This project demonstrates form design, input validation, and clean UI styling.

---

#Features
- Responsive layout centered on the page
- Clean and modern design with shadows and rounded corners
- Input fields for:
  - Full Name
  - Email Address
  - Age (with validation: min 10, max 100)
- Dropdown menu for selecting favorite programming language
- Radio buttons for gender selection
- Checkboxes for skills
- Textarea for additional comments
- Styled submit button with hover effect

---
